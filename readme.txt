File Exchange Program ver 2.2 by Sage Ching-Huai Wang

Summary:
This program copies desinated/specified file to desinated/specified location(especially for internal drives shared upon an organization members), and creates a exact folder structure of the destination drive in current directory (without files). It then copies desinated/specified files to the structure in current folder, then remove empty folders leaving only files under folders/subfolders which is ready to be synced to an offline terminal.
